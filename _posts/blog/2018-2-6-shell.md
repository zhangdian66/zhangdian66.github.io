##字符串分割
```
SplitToArray() {
    local OLD_IFS
    local array
    OLD_IFS="$IFS"
    $IFS="$2"
    array=($1)
    IFS="$OLD_IFS"
    echo ${array[*]}
}
```
这段代码将传入的字符串按照分隔符分割为数组并返回


##参数处理
shell脚本需要处理命令行参数，如下
```
./test.sh -f prog.conf -r --prefix=/home
```
选项:<br>
    - 短选项-f与-v，后面可以带参数
    - 长选项 --prefix
